# Fashionista - E-commerce Product Explorer

Fashionista is a React-based e-commerce web application where users can browse, search, and explore products using a public API.

## Project Description

This project is built using React and Vite. It allows users to:

- Browse product listings
- Search products
- View product details
- Add products to cart
- Remove products from cart
- Login using frontend form validation

The application uses Fake Store API for fetching product data and Context API for cart state management.

## Features

- Product listing using API
- Search functionality
- Product details page
- Cart management
- Login page
- Responsive UI
- React Router navigation
- Context API state management

## Technologies Used

- React
- Vite
- Axios
- React Router DOM
- Context API
- CSS
